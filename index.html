The Archive

The original TextualMachine was an early web experiment: hundreds of handwritten fragments combined through JavaScript to create unexpected relationships between words.

The Idea Behind The Machine

Language is not only created word by word. Meaning emerges through connections, memory, context and unexpected combinations.

TextualMachine explores the space between programming and poetry — where rules create possibilities rather than limitations.
